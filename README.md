# DocHub

Plataforma digital de apoio à carreira médica — serviços essenciais, teste profissional, conteúdo, mentoria e assessorias em um só lugar.

## Tecnologias

- Vite
- TypeScript
- React + React Router
- shadcn/ui
- Tailwind CSS

## Desenvolvimento

Requer Node.js e npm/bun instalados.

```sh
# Instalar dependências
npm install

# Rodar servidor de desenvolvimento
npm run dev

# Build de produção
npm run build

# Testes
npm run test
```
